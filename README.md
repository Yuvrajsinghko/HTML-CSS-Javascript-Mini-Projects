## 🌌 About The Project

> A curated collection of **14 high-quality frontend mini projects** built with pure HTML, CSS, and JavaScript — focused on pushing the boundaries of what's possible in the browser.

From buttery-smooth Lenis scroll experiences to cinematic GSAP animation pipelines, from 3D WebGL experiments to pixel-perfect UI clones — this repository is a **living lab** of modern frontend craftsmanship.

Each project is built to **sharpen real-world skills**, explore new techniques, and ship polished, production-quality UI.

## ⚡ Run Any Project Locally

```bash
# Clone the full repo
git clone https://github.com/Yuvrajsinghko/HTML-CSS-Javascript-Mini-Projects.git

# Enter any project folder
cd HTML-CSS-Javascript-Mini-Projects/GSAP-Wizwebsite

# Open in browser (or use Live Server in VS Code)
open index.html
```

> 💡 **Tip:** Use the **Live Server** VS Code extension for hot-reload while exploring projects.
