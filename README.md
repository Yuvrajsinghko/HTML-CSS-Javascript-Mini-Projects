## 🌌 About The Project

A curated collection of **high-quality frontend mini projects** focused on:

* 🎬 Smooth animations (GSAP)
* 🌊 Fluid scrolling (Lenis)
* 🎯 Micro-interactions & UI polish
* 🧠 Creative frontend experiments

Each project is built to **push UI/UX boundaries** and improve real-world frontend skills.
